PART-2: PROBLEMS WHILE WORKING
<hr>


To start with, 
I had a problem with creating a vertical line to separate my certificates in the past experiences section. That experience was really frustrating because I had the misconception that if a line needs to be drawn, it should only be drawn with the  hr  tag. I wasted so much time trying to make hr  vertical. I later found out that we could use a simple  div <div> tag with CSS and make its background color our vertical  hr 

Next, centering the div <div>, 
Again, in the past experiences section, I have used two of my certificates with different heights and widths if observed precisely. Firstly, I aligned them to the center and was successful. However, when it came to centering the certificates' description, I had a hard time doing it. I was trying to make them center with margin and padding. Then I shifted to justify-content and align-items properties, but they were not willing to center themselves. I wanted to use one property and not overwrite it, keeping my code clean, but both methods were not helpful. In the end, I had to use position: relative and then align it with the left property. These options were there from the start; however, I feel like if nothing seems to work, only then should I use them.
